## Run: 
 ### Windows: 

* get cmake
* get msbuild
```
$ cmake .
$ msbuild JoyconChambara.vcxproj
$ JoyconChambara.exe
```

 ### Linux: 

* get cmake
* get make
```
$ cmake .
$ make
$ ./JoyconChambara
```
