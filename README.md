## Run: 
> ### Windows: 

>* get cmake
>* get mingw
```
$ cmake -G "MinGW Makefiles" .
$ JoyconChambara.exe
```

> ### Linux: 

>* get cmake
>* get make
```
$ cmake .
$ make
$ ./JoyconChambara
```
